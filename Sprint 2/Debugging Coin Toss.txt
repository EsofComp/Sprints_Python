Resolução do Projeto 1 do Capítulo 10:

-> Precisamos achar os erros e conserta-los.
-> O programa dado, traduzindo para português é:

import random
escolha = ''
while escolha not in ('cara', 'coroa'):
    print('Escolha Cara ou Coroa meu bom: ')
    escolha = input()
sorteio = random.randint(0, 1) # 0 é coroa, 1 é cara.
if sorteio == escolha:
    print('Nice, você ganhou!!')
else:
    print('Sorry, vc perdeu!! Tente novamente')
    escolha = input()
    if sorteio == escolha:
       print('Nice, você ganhou!!')
    else:
        print('Desisti, vc é muito Ruim!!')

-> 1º Erro:
Podemos ver que não há um padrão dentro das escolhas, temos somente 'cara' e
'coroa', o nos da apenas duas opções de entrada, logo, o priemiro passo é padro-
nizar.
Sol:

Usamos a função escolha.lower()
>>> escolha = escolha.lower()

-> 2º Erro:
Podemos analisar que nunca conseguimos ganhar, pois esta sorteando um numero, 
e um numero nunca é igual a uma string (nao enquanto ele nao for escrito como 
string)

Sol:
Criamos um if/elif, e uma variavel nova de modo que:
if sorteio == 0:
    sorteado = 'coroa'
elif sorteio == 1:
    sorteado = 'cara'

-> 3º Erro:
Achamos que ao perder, e tentar novamente, vc simplesmente perde por digitar qual
quer coisa que nao seja cara ou coroa, o que é indevido, por ter somente cara
ou coroa, é inviavel a pessoa perder digitando "asgyusifjndfjkg" por exemplo.

Sol:
if (escolha == 'cara' or escolha == 'coroa'):
        if sorteio == escolha:
            print('Nice, você ganhou!!')
        else:
            print('Desisti, vc é muito Ruim!!')
    else:
        print("Por favor, escolha entre as opções fornecidas")

-> 4º Erro:
Ao colocarmos esse termo, vemos que ele nao esta completamente correto, pois ao 
digitar qualquer coisa, voce ve que ele encerra o programa e nao te da a opcao
de tentar novamente.

Sol:
Vamos colocar dentro de um laço.
while escolha not in ('cara' , 'coroa'):
        print ('Nova escolha:')
        escolha = input()
        escolha = escolha.lower()
        if (escolha == 'cara' or escolha == 'coroa'):
            if sorteio == escolha:
                print('Nice, você ganhou!!')
                break
            else:
                print('Desisti, vc é muito Ruim!!')
                break
        else:
            print("Por favor, escolha entre as opções fornecidas")
